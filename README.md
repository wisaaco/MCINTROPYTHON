## How to get the code 

```bash
git clone https://github.com/wisaaco/MCINTROPYTHON.git
cd MCINTROPYTHON
uv sync
```

to install libraries:
```bash
uv add seaborn
```

to rebuild the docs in local:
```bash
 uv run --extra docs sphinx-build -b html docs docs/_build/html
```
